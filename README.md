# 千问网页

这是用于 GitHub Pages 发布的精简版本。

## 发布方式

1. 在 GitHub 新建仓库。
2. 将本文件夹内的全部内容上传到仓库根目录。
3. 在仓库 `Settings > Pages` 中选择从主分支根目录发布。

入口文件为 `index.html`，资源文件统一位于 `assets/media/`。

## 性能优化

- 场景图片已转换为 WebP，并启用延迟加载与异步解码。
- 高码率背景视频已压缩为网页友好的 H.264 MP4。
- 其他背景视频已启用 Fast Start，便于网页渐进加载。
- 所有发布资源使用简洁英文文件名，避免 GitHub Pages 路径兼容问题。
